## 凡例

- 🟢 一貫: 過去の立場と一貫している
- 🟡 部分的に一貫: 部分的に一貫しているが、変更や修正がある
- 🔴 矛盾: 過去の立場と矛盾している
- 🔵 新規: 過去に明確な立場がなく、新しい主張である

## 政党別分析

### 自由民主党（自民党）

**総合評価**: 🔴 矛盾

**公約概要**:
- 食料品の消費税を2年間ゼロに
- 危機管理や成長投資のための「新たな予算枠」
- 衆議院議員定数の1割削減を次期国会で法案成立目標
- 連立合意に基づく「12本の矢」（維新との連立）

**総合分析**:

自民党の2026年衆院選公約は、過去の立場との一貫性において深刻な矛盾を抱えている。特に消費税減税と議員定数削減の2つの主要公約は、過去数年から十数年にわたって党が堅持してきた立場とは正反対の方針転換であり、政権奪取のための戦術的変更と見られる。財政健全化路線との整合性も不明確で、「新たな予算枠」の具体的な財源の裏付けが示されていない。

#### 🔴 食料品の消費税を2年間ゼロに

**評価**: 矛盾

**過去の立場**:

自民党は長年にわたって消費税増税を推進してきた主要政党である。

- **2012年**: 民主党・公明党との三党合意で消費税増税を柱とする社会保障・税一体改革関連法案を成立させ、消費税を8%（2014年4月）、10%（2015年10月、後に延期）へ引き上げることに合意
- **2014年4月**: 安倍内閣下で消費税率を5%から8%に引き上げ実施
- **2019年10月**: 消費税率を8%から10%に引き上げ（食料品等には軽減税率8%を適用）。この際、自民党は「社会保障を担保するために消費税が必要」との立場を明確に示した
- **2022年6月**: 参院選前の市民団体アンケートで、自民党は消費税率引き下げを否定
- **2023-2024年**: 森山幹事長や石破茂首相（当時）が繰り返し消費税減税に反対する発言を行い、「社会保障の重要な安定財源として、消費税率引き下げや廃止を行うことは適当ではない」との立場を維持
- **2024年10月衆院選**: 石破首相は「現時点では税率の引き下げも引き上げも考えていない」と明言

**高市首相の発言のブレ**:

- **2025年5月**: 自民党税制調査会の勉強会後、「国の品格として食料品の消費税率は0%にするべきだ」と強調
- **2025年9月（総裁選）**: 財政規律派の麻生太郎副総裁の支援を得るため、消費税減税の主張を事実上封印
- **2025年10月（首相就任後）**: レジの改修に1年程度かかるとして慎重姿勢を示し、12月のインタビューでも「選択肢としては排除しないが、物価高対策としては即効性がないと判断をした」と発言
- **2026年1月（衆院解散表明）**: 突如として2年間に限り食料品を消費税の対象としない考えを示し、実施を急ぐと方針転換

**分析**:

自民党は長年にわたり「社会保障の安定財源」として消費税の重要性を主張し、増税を推進してきた。2019年の10%への引き上げ時には、低所得者への配慮として軽減税率（8%）を導入したが、あくまで税率維持の枠組みの中での措置だった。

2022年から2024年まで、党幹部は一貫して消費税減税に反対し、野党の減税提案を「財源の裏付けがない」として批判してきた。高市首相自身も、首相就任後は「即効性がない」として慎重姿勢を崩さなかった。

しかし、衆院解散を前に突如として「食料品の消費税2年間ゼロ」を公約に掲げた。この急転換は、維新との連立合意や選挙戦術上の必要性から生じたものと考えられるが、過去30年以上にわたって消費税増税を推進してきた党の立場とは根本的に矛盾している。

また、2026年1月の日経アンケートでは自民党候補の2割が「消費税率現状維持」と回答しており、党内にも食料品の消費税ゼロに対する慎重論が根強く存在することが明らかになっている。公約と党内の実際の意見の乖離も指摘できる。

#### 🔴 衆議院議員定数の1割削減を次期国会で法案成立目標

**評価**: 矛盾

**過去の立場**:

自民党は過去に何度も議員定数削減を公約に掲げながら、実行してこなかった歴史がある。

- **2010年参院選**: 自民党は「2013年までに衆参722人の総定数を650人に、2016年までに500人にする」と公約
- **2012年11月**: 民主党・野田佳彦首相との党首討論で、安倍晋三総裁（当時）は衆議院解散の条件として議員定数削減法案に賛同。民主、自民、公明の三党は「選挙制度の抜本的な見直しについて検討を行い、2013年の通常国会終了までに結論を得た上で必要な法改正を行う」と合意
- **2012年12月**: 衆院解散により比例40削減を含む法案は廃案に。自民党政権復帰後、この約束は実現されず
- **2016年**: 野田聖子議員（当時）が「（自民党が）与党に戻るために約束したことを忘れたのか、忘れたふりをしている」と批判。野田佳彦前首相も「野田氏側が解散という約束を守ったにも関わらず、安倍氏側が定数削減と選挙制度の見直しという約束を守っていない」と指摘
- **2021年衆院選**: 自民党の公約で具体的な定数削減の数値目標は示されず
- **2025年1月**: 自民・維新の協議で、議員定数削減の結論を「2026年以降に」先送りすることで合意していた
- **2025年12月**: 自民党の鈴木俊一幹事長が「衆院小選挙区は難しい」と発言

**実績**:

現行の小選挙区比例代表並立制導入（1994年）以降、定数削減が実現したのは3回のみで、現在の総定数465は戦後最少。しかし、これらは自民党の積極的な削減というより、「一票の格差」是正のための是正措置的な削減が主であった。

**分析**:

自民党は2010年代から繰り返し議員定数削減を「身を切る改革」として公約に掲げてきたが、その実行は極めて限定的だった。特に2012年の三党合意では、野田首相との党首討論で安倍総裁が明確に約束したにもかかわらず、政権復帰後にこれを実行しなかったことは、党内外から強い批判を受けた。

2025年1月には自民・維新の協議で定数削減の結論を「2026年以降に」先送りすることで合意していたにもかかわらず、わずか数か月後の2026年衆院選公約では「次期国会で法案成立を目指す」と前倒しして掲げている。これは明らかに選挙戦術上の方針転換である。

また、鈴木幹事長が「衆院小選挙区は難しい」と述べているように、党内には実現に対する慎重論が根強い。過去14年間にわたって約束を反故にしてきた実績を考えると、今回の公約も選挙のためのリップサービスである可能性が高く、過去の立場との一貫性は皆無である。

#### 🔵 危機管理や成長投資のための「新たな予算枠」

**評価**: 新規（ただし財政健全化路線との整合性に疑問）

**過去の立場**:

自民党は2010年代から「財政健全化」と「プライマリーバランス（基礎的財政収支、PB）黒字化」を掲げてきた。

- **2018年**: 骨太の方針で「2025年度にPB黒字化」を目標として設定
- **2024年6月**: 骨太の方針2024で2025年度PB黒字化目標を堅持
- **2025年1月**: 政府の中期試算で2025年度PBは約4.5兆円の赤字と予測され、黒字化目標の達成困難が明らかに
- **2025年6月**: 自民党財政健全化推進本部が石破首相（当時）に「2025〜26年度にPB黒字化」を提言
- **2025年11月**: 高市首相がPB達成を単年度ごとに確認する方針から複数年で確認する方針へ転換し、「2026年の骨太方針で改めて明確化する」と表明

**現在の公約内容**:

自民党は「責任ある積極財政」の下、経済安全保障、食料、エネルギー、資源、健康医療、国土強靱化、サイバーセキュリティーなどに戦略的に投資するため、「新たな予算枠」を設けることを公約。複数年での機動的な財政出動を可能にするとしている。

**分析**:

「新たな予算枠」自体は過去の自民党政権で明確に提示されていなかった新しい政策であり、🔵新規の主張と評価できる。

しかし、この政策は自民党が長年掲げてきた「財政健全化」「PB黒字化」の目標と根本的に矛盾する可能性がある。高市政権は2025年11月にPB黒字化の確認方法を単年度から複数年に変更しており、事実上の財政健全化目標の後退と見られている。

「新たな予算枠」の具体的な財源については公約で明示されておらず、「責任ある積極財政」という曖昧な表現にとどまっている。食料品の消費税ゼロ化で年間約5兆円の税収減が見込まれる中、さらに「新たな予算枠」で大規模支出を行う財源の裏付けは不明確である。

自民党内の財政規律派（麻生副総裁ら）と積極財政派（高市首相ら）の対立が続く中、この公約は財政規律派の懸念を押し切る形で打ち出された可能性が高い。今後、財源問題や国債市場への影響が大きな争点になると予想される。

#### 🔴 維新との連立政権

**評価**: 矛盾

**過去の立場**:

自民党は1999年から2025年まで、26年間にわたって公明党と連立政権を組んできた。自公連立は日本の政治における最も安定した枠組みの一つと見なされてきた。

- **1999年10月**: 自公連立政権発足
- **2012年12月**: 第2次安倍政権以降、自公連立を基軸とした政権運営が継続
- **2025年夏**: 参院選で公明党が大敗し、自民党への不信が高まる
- **2025年10月**: 高市早苗氏が自民党総裁に就任。公明党は、高市氏が公明嫌いで知られる重鎮や裏金議員を登用し、国民民主党との連携協議を優先させたことに強く反発
- **2025年10月10日**: 公明党が自民党との政策協議で「政治とカネ」問題への対応が不十分として、26年間続いた自公連立を解消

**連立解消の理由**:

公明党は連立の条件として、以下を要求していた：
1. 企業・団体献金の大幅な規制強化
2. 裏金事件の全容解明
3. 靖国神社参拝問題への配慮
4. 外国人との共生の問題への対応

しかし、自民党側は企業団体献金の規制強化について「これから検討する」という不十分な回答に終始し、不記載問題についても全容解明に向けての具体的行動を示さなかった。

**維新との連立合意**:

- **2025年10月20日**: 自民党と日本維新の会が連立政権樹立の合意書に署名
- **2025年10月21日**: 臨時国会の首相指名選挙で維新が高市総裁に投票し、「閣外協力」の形で自維政権が誕生
- **連立合意の内容**: 「日本再起への12本の矢」として、食料品消費税ゼロ、社会保険料引き下げ、副首都構想などを合意

**分析**:

26年間続いた自公連立の解消と維新との新たな連立は、自民党にとって大きな政治的転換である。

公明党との連立解消の最大の理由は「政治とカネ」の問題、特に裏金事件への対応だった。公明党は企業・団体献金の規制強化と事件の全容解明を求めたが、自民党はこれに応じなかった。この点で、自民党は政治改革を重視する公明党の立場を拒否し、より規制に寛容な維新との連携を選択したと言える。

また、高市首相が「公明嫌いで知られる重鎮や裏金議員を登用」したことが公明党の反発を招いた。これは、自民党が政治とカネの問題への真摯な対応よりも、党内の既得権益層への配慮を優先したことを示している。

維新との連立は、消費税減税や副首都構想など、より大胆な政策を打ち出しやすくなる利点がある一方、公明党が長年果たしてきた「歯止め役」を失うことになる。特に靖国神社参拝や外国人政策、憲法改正などの問題で、自民党がより右寄りの政策を進めやすくなる可能性がある。

過去26年間にわたって「自公連立こそが政治の安定」と主張してきた自民党が、裏金問題への対応を回避する形で公明党を切り、維新と組んだことは、党の立場の大きな転換であり、一貫性の観点からは矛盾と評価せざるを得ない。

## 政党横断的な観察

### 1. 消費税減税の政治的潮流

2026年衆院選の特徴は、ほぼ全政党が何らかの消費税減税を公約に掲げている点である。これは過去に例のない現象であり、物価高への国民の不満が極めて強いことを示している。

- **自民党・維新**: 食料品2年間ゼロ
- **中道改革連合（立憲民主+公明）**: 食料品恒久的ゼロ
- **国民民主党**: 一律5%
- **共産党**: 5%への引き下げ（将来的に廃止）
- **れいわ新選組**: 完全廃止

自民党が長年推進してきた消費税増税路線を放棄し、減税に転じたことは、日本の財政政策における大きな転換点となる可能性がある。

### 2. 「身を切る改革」の形骸化

議員定数削減は、2010年代から繰り返し公約に掲げられながら、ほとんど実現してこなかった典型的な「空約束」である。自民党だけでなく、維新も定数削減を掲げているが、過去の実績を見る限り、実現の可能性は極めて低い。

有権者は「身を切る改革」という言葉に惑わされることなく、過去の実績を冷静に評価する必要がある。

### 3. 財源問題の不透明性

自民党の公約では、以下の大規模支出が予定されている：

- 食料品の消費税ゼロ化：約5兆円/年の税収減
- 「新たな予算枠」：規模不明の追加支出
- 社会保険料引き下げ（維新との合意）：約4兆円/年

これらの合計は年間10兆円を超える可能性があるが、財源については「税外収入も活用」といった曖昧な表現にとどまり、具体的な裏付けが示されていない。

プライマリーバランス黒字化目標が事実上後退する中、これらの政策が実現すれば、日本の財政状況はさらに悪化する可能性が高い。国債市場や為替市場への影響も懸念される。

### 4. 政治とカネの問題の未解決

自公連立解消の最大の要因は「政治とカネ」の問題、特に裏金事件への対応だった。公明党は企業・団体献金の規制強化を求めたが、自民党はこれを拒否し、より規制に寛容な維新との連携を選択した。

このことは、自民党が政治改革よりも党内の既得権益層への配慮を優先したことを意味している。選挙後、この問題がどのように扱われるかが注目される。

### 5. 連立政権の枠組み変化がもたらす影響

自公連立から自維連立への転換は、日本の政治に大きな影響を与える可能性がある。公明党は長年、自民党の右寄りの政策に対する「歯止め役」を果たしてきたが、維新はより自民党に近い立場を取ることが多い。

特に以下の分野で政策の右傾化が進む可能性がある：

- 憲法改正
- 靖国神社参拝
- 外国人政策
- 教育政策

一方、規制緩和や地方分権など、維新の政策が自民党政権に取り入れられる可能性もある。

## 結論

自民党の2026年衆院選公約は、過去の立場との一貫性において深刻な問題を抱えている。特に消費税減税と議員定数削減は、過去数十年の党の方針とは正反対の方向転換であり、選挙戦術上の必要性から打ち出されたポピュリズム的公約と見られる。

高市首相の消費税に関する発言のブレ、議員定数削減の過去14年間の不履行、財政健全化目標の事実上の後退など、自民党の公約は整合性と実現可能性の両面で疑問符が付く。

有権者は、選挙公約の内容だけでなく、各党の過去の実績と発言の一貫性を総合的に評価し、投票判断を行うことが重要である。
