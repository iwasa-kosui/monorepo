# 中道改革連合 公約一貫性分析レポート

**選挙日**: 2026年2月8日

## 凡例

- 🟢 一貫: 過去の立場と一貫している
- 🟡 部分的に一貫: 部分的に一貫しているが、変更や修正がある
- 🔴 矛盾: 過去の立場と矛盾している
- 🔵 新規: 過去に明確な立場がなく、新しい主張である

## エグゼクティブサマリー

中道改革連合は、2026年1月16日に立憲民主党と公明党の衆議院議員（約172人）が合流して結成された新党である。イデオロギー的に異なる両党の合流は、2025年10月の自公連立解消という政局的変動を背景としており、主要公約である「食料品消費税の恒久的ゼロ」については、両党の過去の立場と比較して重大な矛盾や転換が認められる。

## 政党別分析

### 立憲民主党（中道改革連合の母体の一つ）

**総合評価**: 🔴 矛盾

**公約概要**: 食料品消費税の恒久的ゼロ、若者・学生向け賃貸住宅の家賃補助・住宅提供、中道路線による政権交代

**総合分析**: 野田佳彦代表は2012年に消費税10%への引き上げを主導した財政規律派の象徴的存在であり、2024年の代表選でも「安易な消費税減税はしない」と明言していた。しかし、党内の減税派の圧力と選挙対策を優先し、2025年4月に「時限的な食料品消費税ゼロ」に転換、さらに2026年1月の新党結成時には「恒久的ゼロ」へと政策を拡大させた。この転換は野田代表の政治信条と真っ向から対立するものであり、明確な矛盾が認められる。

#### 🔴 食料品消費税の恒久的ゼロ

**評価**: 矛盾

**過去の立場**:

- **2012年**: 野田佳彦首相（当時）は自民・公明との三党合意で消費税率を段階的に10%に引き上げることを決定し、「政治生命を賭ける」と表明した
- **2024年8月**: 立憲民主党代表選で野田氏は「消費税を基幹税と位置付けることで、日本社会は成り立っている」「安易に減税はできない」と明言
- **2024年末**: 党内の減税派（江田憲司氏ら）が消費税減税の勉強会を立ち上げたが、枝野幸男元代表は「減税ポピュリズムに走りたいのなら、別の党をつくるべきだ」と批判
- **2025年2月**: 党内に「財政規律」路線への不協和音が広がり、有志議員が食料品消費税「当面ゼロ」実現に向けた勉強会を開始（参加者は党所属国会議員の3割超）

**現在の立場**:

- **2025年4月25日**: 野田代表が苦渋の決断として、食料品の消費税を時限的（1年間、最大2年延長可能）にゼロにする公約を採用
- **2026年1月16日**: 中道改革連合結成時に「食料品消費税の恒久的ゼロ」に政策を拡大
- 財源として「ジャパンファンド」構想を提示（政府保有資産500兆円を運用し、利回り1%改善で5兆円捻出）

**分析**:

野田佳彦代表の立場は完全に逆転している。「消費税10%への引き上げを政治生命を賭けて実現した人物」が、「消費税恒久的ゼロ」を掲げるに至った経緯は、政策の一貫性よりも党内融和と選挙対策を優先したものと評価せざるを得ない。

特に注目すべきは転換の段階性である。当初は「時限的（1年間）」だったものが、公明党との合流により「恒久的」へと拡大している。これは公明党の政策に立憲民主党が歩み寄った結果であり、野田代表の本来の政治信条である財政規律とは相容れない。

また、「ジャパンファンド」構想による財源確保案は実現可能性に疑問が残り、「年金ドロボー」との批判も噴出している。給付付き税額控除が本来の目標であるとの説明も、恒久的ゼロ税率との整合性が不明確である。

#### 🟡 中道路線への転換

**評価**: 部分的に一貫

**過去の立場**:

- 立憲民主党は結成当初（2020年）、リベラル政党として評価されていた
- 2022年の参院選までは「分配」を強調し、左派的な性格が強かった
- 野田佳彦代表は自身の立場を「中道から穏健な保守」と位置付けており、立憲民主党内では相対的に保守寄りの立場だった

**現在の立場**:

- 2024年衆院選から「成長」の重みが増加し、やや右側へシフト
- 2026年1月の公明党との合流により、「中道路線」を明確化
- 「イデオロギーや対立を優先する政治ではなく、生活者一人ひとりの現実から出発する政治」を掲げる

**分析**:

立憲民主党の中道路線への転換は、野田代表個人の政治的立場とは整合的である。野田氏はもともと党内では保守派・財政規律派として知られており、「中道から穏健な保守」を自認していた。

しかし、党全体としては明確な路線転換である。立憲民主党は2024年衆院選で得票が伸び悩み、党内で「中道」の立ち位置を再確認する総括をまとめたが、依然として党内の2割程度を占める「リベラル派」が存在し、路線転換には抵抗が残っている。

公明党との合流は、この中道路線を制度的に確定させる意味を持つが、立憲民主党内のリベラル派にとっては受け入れがたい転換である可能性がある。

### 公明党（中道改革連合の母体の一つ）

**総合評価**: 🔴 矛盾

**公約概要**: 食料品消費税の恒久的ゼロ、若者・学生向け賃貸住宅の家賃補助・住宅提供、中道路線による政権交代

**総合分析**: 公明党は2019年に政党として唯一、軽減税率の導入を実現した実績を持ち、「恒久的な措置」を一貫して主張してきた。しかし、その主張は「軽減税率8%を5%程度に引き下げる」ものであり、「ゼロ」ではなかった。また、時限的な減税には明確に反対し、「物価高対策のために一時的に税率を下げることは適切ではない」との立場を示していた。中道改革連合での「恒久的ゼロ」公約は、恒久措置という点では一貫しているが、「ゼロ」という水準は過去の主張を大きく超えている。さらに、26年間の自公連立を解消して立憲民主党と合流したこと自体が、イデオロギー的に重大な転換である。

#### 🟡 食料品消費税の恒久的ゼロ

**評価**: 部分的に一貫

**過去の立場**:

- **2019年10月**: 公明党が主導し、食料品などの軽減税率8%を実現（酒類・外食を除く飲食料品全般）
- **2024年**: 公明党は軽減税率を「福祉的な観点」で引き下げることを主張し、「恒久的な措置にすべきだ」との立場を表明
- **2025年5月**: 斉藤鉄夫代表が「軽減税率を例えば5%にする」との一案を示し、「恒久的措置として考えている」と説明
- **2025年5月（主張）**: 「消費税は社会保障を支える重要な安定財源として不可欠。物価高対策のために一時的に税率を下げることは適切ではない」と時限的減税に明確に反対

**現在の立場**:

- **2026年1月16日**: 中道改革連合結成時に「食料品消費税の恒久的ゼロ」を公約として採用
- インボイス制度の廃止も公約に含まれる

**分析**:

公明党の立場は「部分的に一貫」と評価できる。軽減税率の「恒久的措置」という基本方針は一貫しているが、その水準が「8%維持→5%程度→0%」へと段階的に拡大している。

特に注目すべきは、公明党が一貫して反対してきた「時限的措置」ではなく、「恒久的措置」を中道改革連合でも主張している点である。これは公明党の基本的な政策哲学（社会保障財源の安定性）と整合的である。

ただし、「0%」という水準は過去の主張を大きく超えるものである。2025年5月時点では「5%程度」が想定されており、わずか8か月後に「0%」へと政策を拡大したことになる。これは立憲民主党との合流交渉において、立憲側の主張に歩み寄った結果と考えられる。

また、公明党は2025年の参院選では消費税減税を「公約ではない」と明言しながらも「公約集に課題として載せる」と補足しており、慎重な姿勢を維持していた。中道改革連合での「恒久的ゼロ」は、この慎重姿勢からの大きな転換である。

#### 🔴 自公連立の解消と立憲民主党との合流

**評価**: 矛盾

**過去の立場**:

- **1999年10月～2025年10月**: 公明党は26年間、自民党と連立政権を維持
- 公明党は「中道」を掲げてきたが、その意味は「人間中心主義」（左右のイデオロギー対立の中間ではなく、人々の幸福と平和を最優先に考える姿勢）
- 公明党は平和主義を重視し、創価学会の支持基盤を持つ

**連立解消の経緯（2025年10月）**:

- **2025年10月10日**: 斉藤鉄夫代表が高市早苗自民党総裁に連立解消を通告
- 連立解消の理由として以下の3点を挙げた：
  1. 政治とカネの問題の全容解明（裏金問題への自民党の消極的姿勢）
  2. 靖国神社参拝を含む歴史認識（高市氏の靖国参拝への懸念）
  3. 過度な外国人排斥（高市氏の排外的姿勢への懸念）
- 企業・団体献金の規制強化を巡り、公明党は受け皿を党本部・都道府県支部連合会・国会議員支部に限定する案を提示したが、自民党は反発

**現在の立場**:

- **2026年1月15日**: 野田代表と斉藤代表が党首会談を実施
- **2026年1月16日**: 立憲民主党と公明党が合流し、中道改革連合を結成（衆議院限定、約172人）
- 新党は「生活者ファースト」を掲げ、イデオロギー対立よりも現実的政策を重視

**分析**:

公明党の立憲民主党との合流は、イデオロギー的に重大な転換である。

公明党が自民党との連立を解消した理由は、高市早苗総裁の「右傾化」への懸念（靖国参拝、外国人排斥）と、裏金問題への自民党の消極的姿勢であった。特に靖国参拝問題は、反戦・平和主義を掲げる創価学会の思想的根幹に関わる問題であり、「国家神道批判から出発した創価学会」にとって「思想の根本が交わらない」問題だった。

しかし、立憲民主党もまた公明党とはイデオロギー的に異なる政党である。立憲民主党は従来、リベラル政党として憲法9条改正反対、反原発、選択的夫婦別姓推進など、左派的な政策を掲げてきた。一方、公明党は自民党との連立下で、これらの政策には慎重ないし反対の立場を取ってきた。

両党の「中道」の意味も異なる。立憲民主党（特に野田代表）の「中道」は政治的スペクトラムの中間（保守とリベラルの中間）を意味するのに対し、公明党の「中道」は「人間中心主義」を意味する。このイデオロギー的差異を「生活者ファースト」という曖昧な概念で覆い隠していると言わざるを得ない。

また、26年間の同盟関係にあった自民党との連立を解消し、わずか3か月後に政敵であった立憲民主党と合流したことは、支持者にとって理解困難な転換である。高市氏への懸念が連立解消の主因であったとしても、それが立憲民主党との合流を正当化する理由となるかは疑問である。

## 政党横断的な観察

### 1. 政局優先の公約形成

中道改革連合の公約、特に「食料品消費税の恒久的ゼロ」は、両党の過去の立場からの大きな転換である。この転換は、政策の整合性や実現可能性よりも、以下の政局的要因によって推進されたと考えられる：

- **立憲民主党**: 党内の減税派の圧力、選挙での支持拡大の必要性、公明党との合流交渉での譲歩
- **公明党**: 高市政権への対抗、自民党との連立解消後の新たな連携先の確保、選挙での生き残り

両党とも、従来の政策的立場を曲げてでも、政権交代の可能性を追求する選択をしたと言える。

### 2. 「中道」概念の曖昧さ

立憲民主党と公明党が掲げる「中道」は、それぞれ異なる意味を持つ：

- **立憲民主党（野田代表）の中道**: 政治的スペクトラムの中間、保守とリベラルの中庸、財政規律の重視
- **公明党の中道**: 人間中心主義、イデオロギー対立の超越、平和主義と福祉重視

この概念的差異を「生活者ファースト」という包括的なスローガンで統合しているが、具体的な政策場面で両者の「中道」が衝突する可能性は高い。

### 3. イデオロギーよりも反高市で結束

両党の合流の最大の要因は、高市早苗政権への対抗という「共通の敵」である。公明党は高市氏の右傾化（靖国参拝、外国人排斥）と裏金問題への対応に不信感を抱き、立憲民主党は政権交代の機会を見出した。

しかし、「反高市」という消極的動機に基づく連携は、高市政権が倒れた後の求心力を欠く可能性がある。両党の政策的・イデオロギー的差異は依然として存在し、連立政権を樹立できたとしても、政策実行段階で深刻な対立が生じる恐れがある。

### 4. 財源問題の不透明性

中道改革連合の「食料品消費税恒久的ゼロ」は年間5兆円規模の財源を必要とするが、「ジャパンファンド」構想による財源確保案は以下の点で実現可能性に疑問が残る：

- 政府保有資産500兆円の運用利回りを1%改善させることが本当に可能か
- 政府系ファンドの運用リスクをどう管理するか
- 既存の特別会計や基金の剰余金を取り崩すことの持続可能性
- 「年金ドロボー」との批判への対応

野田代表がかつて重視していた財政規律の観点からも、この財源計画は説得力に欠ける。

### 5. 支持基盤の混乱

両党の合流は、それぞれの支持基盤に混乱をもたらす可能性がある：

- **立憲民主党支持者**: リベラル派は中道化・保守化に反発、野田代表の消費税政策転換に戸惑い
- **公明党支持者（創価学会員）**: 26年間の自民党との同盟関係解消への困惑、立憲民主党との合流への違和感
- **無党派層**: 両党の政策転換を「政局優先」と見なし、信頼性に疑問

特に創価学会の会員には保守的な価値観を持つ人も多く、リベラル政党との合流に抵抗感を持つ層が一定数存在すると考えられる。

### 6. 選挙情勢と公約の実現可能性

日経新聞の終盤情勢調査によれば、自民党＋維新は300議席超をうかがい、中道改革連合は公示前167議席から半減の可能性がある。この厳しい情勢は、以下を示唆する：

- 有権者は中道改革連合の公約や合流を評価していない
- 「食料品消費税恒久的ゼロ」は実現の見込みが薄い
- 政局優先の公約形成が裏目に出た可能性

## 結論

中道改革連合の主要公約である「食料品消費税の恒久的ゼロ」は、立憲民主党・公明党両党の過去の立場と比較して、重大な矛盾や転換が認められる。

**立憲民主党**については、野田佳彦代表が2012年に消費税10%引き上げを主導し、2024年まで「安易な減税はしない」と財政規律を重視していたにもかかわらず、党内の圧力と選挙対策により政策を180度転換させた点で、明確な矛盾がある。

**公明党**については、軽減税率の「恒久的措置」という方針は一貫しているものの、その水準が「8%→5%→0%」へと急速に拡大した点、および26年間の自公連立を解消して立憲民主党と合流した点で、過去の立場からの大きな転換がある。

両党の合流は、高市早苗政権への対抗という政局的動機が主因であり、政策的・イデオロギー的な整合性は二の次になっている。「中道」という概念も両党で意味が異なり、具体的な政策実行段階で対立が生じる可能性が高い。

有権者は、この公約の一貫性の欠如を厳しく評価しており、選挙情勢は中道改革連合に厳しいものとなっている。政策の信頼性よりも政局を優先した結果、かえって支持を失った可能性がある。
