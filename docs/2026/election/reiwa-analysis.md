# れいわ新選組 2026年衆院選 公約一貫性分析レポート

**選挙日**: 2026年2月8日
**生成日時**: 2026-02-08T00:00:00+09:00

## 凡例

- 🟢 一貫: 過去の立場と一貫している
- 🟡 部分的に一貫: 部分的に一貫しているが、変更や修正がある
- 🔴 矛盾: 過去の立場と矛盾している
- 🔵 新規: 過去に明確な立場がなく、新しい主張である

## 政党別分析

### れいわ新選組

**総合評価**: 🟢 一貫

**公約概要**:
- 消費税の完全廃止
- 物価高対策として現金10万円の一律給付

**総合分析**:

れいわ新選組は、2019年の結党時から2026年の現在まで、極めて高い政策一貫性を維持している。「消費税廃止」と「現金給付」という2つの柱は、結党以来の核心的政策であり、一度もブレることなく主張し続けている。

ただし、2026年の公約は従来の「8つの緊急政策」から大幅に絞り込まれており、最低賃金1500円、奨学金徳政令、住宅政策などの重要政策が表面上は言及されていない。これは政策の変更ではなく、今回の選挙戦略として「消費税廃止」と「10万円給付」という2点に絞り込んだメッセージングを行っているものと考えられる。

---

#### 🟢 消費税の完全廃止

**評価**: 一貫

**過去の立場**:

- **2019年4月（結党時）**: 「8つの緊急政策」の第1項目として「消費税は廃止」を掲げて結党
- **2019年7月（第25回参院選）**: 公約の筆頭に「消費税は廃止」を明記し、この政策を前面に押し出して選挙戦を展開
- **2021年10月（第49回衆院選）**: 「れいわニューディール」において消費税廃止を継続して主張
- **2022年7月（第26回参院選）**: 物価高対策の3本柱の1つとして「消費税廃止」を掲げる
- **2024年10月（第50回衆院選）**: 消費税廃止を前面に押し出した公約を発表
- **2025年6月（第27回参院選）**: 消費税の即時全廃を公約の中心に据える
- **2026年2月（第51回衆院選）**: 消費税の完全廃止を継続して主張

**分析**:

れいわ新選組の消費税廃止政策は、結党から7年間、一度も揺らぐことなく主張され続けている。山本太郎代表は結党記者会見（2019年4月10日）の時点から一貫して消費税廃止を訴えており、この政策は党のアイデンティティそのものと言える。

消費税を「5%に減税」ではなく「完全廃止」とする点も、他党との差別化を明確にしており、結党時から現在まで全くブレていない。この一貫性は、れいわ新選組が政権交代の現実性よりも、理念と原則を重視する政党であることを示している。

#### 🟢 物価高対策として現金10万円の一律給付

**評価**: 一貫

**過去の立場**:

- **2020年4月（コロナ初期）**: 政府の特別定額給付金（10万円）について、山本太郎代表は全国民への一律給付を強く支持し、当初の「30万円を限定世帯に」という政府案を批判
- **2020年8月**: 「コロナ緊急政策と財源」において、継続的な現金給付の必要性を訴える
- **2022年7月（第26回参院選）**: 「季節ごとに一律10万円給付」を公約の3本柱の1つとして掲げる
- **2024年10月（第50回衆院選）**: 消費税廃止と10万円給付を公約の柱として明記
- **2025年6月（第27回参院選）**: 物価高対策として国民に現金10万円を一律給付すると公約
- **2026年2月（第51回衆院選）**: 物価高対策として現金10万円の一律給付を継続

**分析**:

れいわ新選組の現金給付政策も、高い一貫性を示している。コロナ禍の2020年、政府が当初「生活困窮世帯に限定して30万円」を給付する方針だった際、山本太郎代表は「全国民への一律給付」を強く主張し、最終的に政府が方針転換して「全国民に10万円」となった経緯がある。

その後、れいわ新選組は「季節ごとに10万円」「一律10万円」という形で、定期的かつ普遍的な現金給付を一貫して主張してきた。この政策は、MMT（現代貨幣理論）に基づく積極財政論と整合しており、「財源は国債発行で賄う」という立場も変わっていない。

「一律給付」という点も重要で、所得制限を設けず全国民に給付するという方針は、結党時からの「ボトムアップ型経済」「全ての人々の生活保障」という理念と完全に一致している。

## MMT（現代貨幣理論）に基づく財政政策との一貫性

### 🟢 積極財政論の一貫性

**評価**: 一貫

**過去の立場**:

山本太郎代表は、結党以前から「MMT（現代貨幣理論）」を支持し、「通貨発行権を持つ国は、インフレにならない限り国債をいくら発行しても問題ない」という立場を取ってきた。地方遊説でMMT関連書籍を「現代の必読書」として公言するなど、MMTを理論的基盤としてきた。

**分析**:

れいわ新選組の全ての経済政策（消費税廃止、現金給付、最低賃金1500円、奨学金徳政令など）は、MMTに基づく積極財政論で一貫している。「財源は国債発行で賄う」という立場は結党以来変わっておらず、これは他の主要政党とは一線を画す独自の経済理論的立場である。

この理論的一貫性により、個々の政策だけでなく、政策パッケージ全体として整合性が取れている。一方で、MMTは多くの経済学者から批判されており、「インフレになっても止められる」という前提が「根拠のない、非現実的な楽観論」と指摘されている点は留意が必要である。

## 政策の絞り込みについて

### 2026年の戦略的メッセージング

2026年の衆院選では、れいわ新選組は公約を「消費税の完全廃止」と「現金10万円の一律給付」という2点に絞り込んでいる。これは政策の変更ではなく、メッセージングの戦略的絞り込みと考えられる。

**理由**:
1. 党の公式サイトや詳細公約には、従来の政策（最低賃金1500円、奨学金徳政令、住宅政策など）が引き続き掲載されている
2. 消費税廃止と現金給付は、有権者にとって最も分かりやすく、即効性のある政策である
3. 短期決戦の選挙戦において、シンプルなメッセージで差別化を図る戦略と考えられる

**評価**: この絞り込みは、政策の変更や後退ではなく、選挙戦術としての判断であり、一貫性を損なうものではない。

## 政党横断的な観察

### 消費税減税をめぐる政党間の違い

2026年衆院選では、ほぼ全ての政党が何らかの消費税減税を掲げているが、その内容には大きな差がある。

| 政党 | 消費税政策 | 特徴 |
|------|-----------|------|
| 自民党・維新 | 食料品の消費税を2年間ゼロ | 期間限定・品目限定 |
| 中道改革連合 | 食料品消費税の恒久的ゼロ | 恒久的だが品目限定 |
| 国民民主党 | 消費税を一律5%に引き下げ | 期間限定・全品目 |
| 共産党 | 緊急に5%、将来的に廃止 | 段階的アプローチ |
| **れいわ新選組** | **消費税の完全廃止** | **即時・全品目・恒久的** |

れいわ新選組の「完全廃止」は、最も徹底した立場であり、他党との明確な差別化となっている。これは結党時から一貫した立場であり、他党が消費税減税を公約に加える中でも、れいわ新選組だけが「廃止」を主張し続けている点は注目に値する。

### 現金給付をめぐる政党間の違い

| 政党 | 給付政策 | 対象 |
|------|---------|------|
| 自民党 | 明示なし | - |
| 中道改革連合 | 若者・学生向け家賃補助 | 限定的 |
| 国民民主党 | 基礎控除178万円（実質的な給付） | 納税者 |
| **れいわ新選組** | **10万円一律給付** | **全国民** |
| 参政党 | 子ども1人あたり月10万円 | 子育て世帯 |

れいわ新選組の「全国民への一律給付」は、所得制限を設けない普遍的な給付であり、これも結党時からの理念と一致している。

## 結論

れいわ新選組の2026年衆院選公約は、結党以来7年間にわたって極めて高い一貫性を維持している。

**一貫している点**:
- 消費税の完全廃止（2019年～2026年）
- 全国民への一律現金給付（2020年～2026年）
- MMTに基づく積極財政論（結党前～2026年）
- 最低賃金1500円、奨学金徳政令などの基本政策（2019年～2026年）

**評価できる点**:
- 理念と原則を貫く姿勢
- 他党に流されない独自路線
- 経済理論的な整合性

**懸念される点**:
- MMTの理論的妥当性に対する批判
- 財源論の現実性
- 政権交代の可能性を考慮しない純粋野党的立場

総合的に見て、れいわ新選組は日本の主要政党の中で最も政策一貫性の高い政党の1つと評価できる。これは、政権交代の現実性よりも、理念と原則を重視する政党としての特性を反映している。
