## 凡例

- 🟢 一貫: 過去の立場と一貫している
- 🟡 部分的に一貫: 部分的に一貫しているが、変更や修正がある
- 🔴 矛盾: 過去の立場と矛盾している
- 🔵 新規: 過去に明確な立場がなく、新しい主張である

## 政党別分析

### 国民民主党

**総合評価**: 🟢 一貫

**公約概要**: 国民民主党は「手取りを増やす」というスローガンのもと、消費税の5%への一律減税、基礎控除の178万円への引き上げ、社会保険料の還付制度創設、年間5兆円の教育国債発行、家賃控除制度と空室税の導入を掲げている。

**総合分析**: 国民民主党の2026年衆院選公約は、概ね過去の主張と一貫している。玉木雄一郎代表は2019年頃から消費税減税を主張し続けており、2021年の衆院選から現在まで「消費税5%への減税」「教育国債の発行」を一貫して公約に掲げている。「103万円の壁」の引き上げについても、2024年衆院選で初めて明確に「178万円」という数値目標を掲げ、2025年末に自民党との政策協議で実現にこぎつけた。社会保険料の還付制度と住宅政策（空室税・家賃控除）は2024年〜2025年にかけて新たに追加された政策だが、いずれも「手取りを増やす」という一貫した理念に基づいている。ただし、玉木代表は消費税減税の実施条件（賃上げ状況）について、2025年7月に一時「今の賃上げ水準なら不要」と述べるなど、情勢に応じた発言のブレも見られた。

#### 🟢 消費税を一律5%に引き下げ（賃金上昇率が物価+2%に安定するまで）

**評価**: 一貫

**過去の立場**:

- **2019年11月**: 玉木代表は毎日新聞のインタビューで「消費を冷やすような税制は控えたほうがよい」と述べ、消費税の廃止も含めて議論する考えを示した
- **2020年**: 新型コロナウイルス感染拡大下で、玉木代表はX（旧Twitter）で「日本も消費税の減税を行うべきだ」と訴え、「消費税減税（10%→5%）」や「半年間0%なども検討したい」と投稿
- **2021年衆院選**: 国民民主党は公約で「賃金上昇率が物価＋2％に達するまで消費税5％に減税する」と明記
- **2024年9月**: 国民民主党は「2024年重点政策」として消費税減税をパッケージに含めた
- **2025年7月1日**: 玉木代表は「いまの数字だと消費減税をしてまで景気を刺激する状況にはない」と発言
- **2025年7月2日**: トランプ米大統領の関税交渉発言を受け、玉木代表は「不確実性がこれまで以上に高まった。やるべきだ」と消費税減税を推進する考えを示した
- **2026年1月**: 玉木代表は衆院選に際し、「中小企業の賃上げがこの春闘で5%程度になるかが最終的な判断基準になる」と説明

**分析**: 国民民主党は2019年頃から一貫して消費税減税を主張しており、特に2021年衆院選以降は「賃金上昇率が物価+2%に達するまで」という条件付きで「消費税5%への減税」を継続的に公約に掲げている。玉木代表の経済政策の根幹は「名目賃金上昇率が、物価上昇率＋2%（当面の間5%）になるまで、財政政策や金融政策を積極的に活用する」というもので、この枠組みは変わっていない。ただし、2025年7月には賃上げ状況に応じて「不要」から「やるべき」へと短期間で立場を変えるなど、実施時期の判断については情勢に応じた変化が見られる。これは矛盾というよりも、条件を重視する姿勢の表れと解釈できる。

#### 🟢 基礎控除を178万円に引き上げ（年収の壁問題の解消）

**評価**: 一貫

**過去の立場**:

- **2024年10月**: 国民民主党は衆院選で「年収の壁を178万円まで引き上げる」と公約に明記。178万円という数値は、1995年に103万円の壁が設定されて以降の最低賃金の上昇率1.73倍を反映させたもの（103万円×1.73倍=178万円）
- **2024年10月27日**: 衆院選で国民民主党が28議席を獲得し、自民・公明与党が過半数割れとなったことで、玉木代表の政策が政権運営を左右する存在に浮上
- **2024年11月5日**: 玉木代表は立憲民主党の野田代表と会談し、基礎控除など所得税の非課税枠拡大に向けて協議を行う方針を確認
- **2024年11月**: 玉木代表はX（旧Twitter）で「いわゆる『103万円の壁』、少し話を整理します」と長文を投稿し、基礎控除は憲法25条の「生存権」を保障するための制度であり、インフレで生きるコストが上がっているのに1995年の水準のままでは生活が苦しくなると説明
- **2025年12月18日**: 高市早苗首相と玉木代表が会談し、年収の壁を178万円まで引き上げることに正式合意

**分析**: 国民民主党は2024年衆院選で初めて明確に「178万円」という数値目標を掲げた。それ以前は「年収の壁」問題について言及していたものの、具体的な数値目標は示していなかった。したがって、この政策は2024年に本格的に打ち出された「新しい主張」と見なすこともできるが、「手取りを増やす」という党の基本理念、および最低賃金の上昇率という客観的な指標に基づいた合理的な提案であり、一貫性を持っている。2024年10月の衆院選で掲げた公約を、2025年末に自民党との政策協議で実現させており、政策の継続性と実現力を示している。

#### 🔵 社会保険料の還付制度創設

**評価**: 新規

**過去の立場**:

- **2024年9月20日**: 国民民主党が「2024年重点政策」を発表し、所得税・消費税の減税とともに社会保険料の軽減を政策パッケージに含めた
- **2024年10月（衆院選公約）**: 国民民主党は「社会保険料の還付制度を創設」し「現役世代の負担を軽減する」ことを公約に掲げた。具体的には、賃上げを行う中小企業・零細企業の事業主の社会保険料を半減させるとした
- **玉木代表の説明**: 「税金・社会保険料が高いため、給料が上がっても手取りが増えない」という問題を指摘し、「社会保険料負担分を給付という形で還付してあげることで、社会保険料負担を速やかに現実的に減免する政策を進めたい」と述べた

**分析**: 社会保険料の還付制度は、2024年に新たに打ち出された政策である。それ以前は「社会保険料の軽減」という一般的な方向性は示されていたが、「還付制度」という具体的な仕組みは明示されていなかった。この政策は、国民民主党の「手取りを増やす」という一貫した理念に基づいており、所得税減税や消費税減税と並んで、現役世代の可処分所得を増やすための新たなアプローチとして位置づけられている。新規の政策ではあるが、党の基本方針との整合性は高い。

#### 🟢 年間5兆円の「教育国債」発行、高校までの教育費完全無償化

**評価**: 一貫

**過去の立場**:

#### 🔵 家賃控除制度の新設、「空室税」導入

**評価**: 新規

**過去の立場**:

- **2025年7月2日**: 国民民主党は第27回参議院議員通常選挙の追加公約として「住宅確保支援策の拡充」を発表。中低所得者を対象にした家賃控除制度の新設と、投資目的の不動産売買を抑制する「空室税」の導入を明記
- **背景**: 東京23区内では中古マンション価格が1億円超、平均家賃（シングル向きマンション）が10万円超となるなど、特に大都市の住宅価格の高騰が顕著。外国人による不動産取得がこの上昇の一因とされている
- **2025年12月11日**: 国民民主党は議員立法「非居住住宅税及び超短期所有住宅等の譲渡に係る事業所得等の課税の特例の創設等に関する法律案」（空室税法案）を衆議院に提出
- **法案内容**: 住宅価格・家賃が投機的取引で高騰した地域（東京23区、指定都市、リゾート地など）を対象に、居住目的以外の住宅に対して市町村が「非居住住宅税」を課税できるようにする。また、住宅用不動産を取得後2年以内に譲渡した場合、個人は分離課税、法人は追加課税を行う（転勤・医療・介護などやむを得ない場合は除外）

**分析**: 家賃控除制度と空室税は、2025年の参院選で新たに追加された政策である。それまで国民民主党が住宅政策について具体的な提案をしていた形跡はなく、明らかに新規の政策である。ただし、この政策も「手取りを増やす」という党の基本理念（家賃控除による可処分所得の増加）と、住宅価格高騰への対応（空室税による投機抑制）という現実的な問題意識に基づいており、党の政策体系の中で整合性を持っている。2025年7月に追加公約として発表後、同年12月には議員立法として法案提出まで行っており、実現に向けた具体的な動きも見られる。

## 政党横断的な観察

### 国民民主党の政策理念の一貫性

国民民主党の2026年衆院選公約は、「手取りを増やす」という明確な理念のもとに体系化されている。消費税減税、基礎控除引き上げ、社会保険料還付、家賃控除という各政策は、すべて「可処分所得の増加」という共通の目標に向かっており、政策パッケージとしての一貫性が高い。

### 玉木代表の経済思想の根幹

玉木代表の経済政策の根幹は「名目賃金上昇率が、物価上昇率＋2%（当面の間5%）になるまで、財政政策や金融政策を積極的に活用する」というもので、2019年頃からこの考え方は一貫している。消費税減税も、この経済政策の枠組みの中で「賃金上昇が不十分な間の時限措置」として位置づけられており、理論的な一貫性がある。

### 「新規政策」の追加と政策体系の進化

社会保険料の還付制度（2024年）、家賃控除・空室税（2025年）は新たに追加された政策だが、いずれも「手取りを増やす」という基本理念に基づいており、政策体系の拡充と見なせる。国民民主党は、基本理念を維持しながら、現実の経済・社会状況（賃上げ動向、住宅価格高騰など）に応じて具体的な政策を追加・調整している。

### 玉木代表の発言のブレと「条件重視」の姿勢

消費税減税について、玉木代表は2025年7月に「今の賃上げ水準なら不要」から「やるべき」へと短期間で立場を変えた。これは一見「ブレ」に見えるが、「賃金上昇率が物価+2%に達するまで」という条件を重視する姿勢の表れとも解釈できる。玉木代表は経済指標や国際情勢に応じて柔軟に判断を変えており、これを「矛盾」と見るか「現実主義」と見るかは評価が分かれる。

### 議員立法と政策実現力

国民民主党は、教育国債法案（2022年）、空室税法案（2025年）、そして基礎控除引き上げの与党協議（2025年）など、公約を単なるスローガンに終わらせず、議員立法や政策協議を通じて実現を図っている。2024年衆院選で獲得した28議席という立場を活用し、与野党双方に政策実現を迫る「キャスティングボート戦略」を展開している。

### 財源論の透明性と課題

国民民主党は、教育国債、日銀保有国債の永久国債化、特別会計の活用など、増税に頼らない財源確保の手段を具体的に示している。これは財源論の透明性を高めている一方で、「赤字国債ではない」という理屈が財政規律の観点から妥当かどうかは、経済学者や財政専門家の間でも評価が分かれる。国民民主党の財政政策は「積極財政」の立場を明確にしているが、中長期的な財政持続可能性についてはさらなる議論が必要である。

## 総括

国民民主党の2026年衆院選公約は、過去の主張と概ね一貫している。特に「消費税5%への減税」「教育国債の発行」は2021年衆院選以降の継続的な公約であり、玉木代表の経済政策の根幹である「賃金上昇率が物価+2%に達するまで財政・金融政策を活用する」という理念は2019年頃から変わっていない。「基礎控除の178万円への引き上げ」は2024年に具体化された政策だが、最低賃金の上昇率という客観的な指標に基づいており、合理性がある。「社会保険料の還付制度」「家賃控除・空室税」は新規の政策だが、「手取りを増やす」という党の基本理念に基づいており、政策体系の拡充と見なせる。

一方で、玉木代表は消費税減税の実施時期について、賃上げ状況や国際情勢に応じて発言を変えており、「ブレ」との批判も受けている。これは「条件重視」の姿勢とも解釈できるが、政治的な駆け引きや情勢判断の影響も否定できない。

全体として、国民民主党の公約は「手取りを増やす」という明確な理念のもとに体系化されており、政策の一貫性と実現力を兼ね備えていると評価できる。ただし、財源論の妥当性や中長期的な財政持続可能性については、引き続き検証が必要である。
