# 第51回衆議院議員総選挙 障害者福祉・障害者の権利に関する政党別政策分析

**選挙日**: 2026年2月8日（第51回衆議院議員総選挙）
**生成日時**: 2026-02-08T00:00:00+09:00
**分析テーマ**: 障害者福祉・障害者の権利（精神障害者・発達障害者に重点）

## 分析の背景

### 障害者福祉をめぐる近年の主要動向

1. **障害者権利条約（CRPD）対日審査**（2022年9月）: 国連障害者権利委員会が日本政府に対し93項目の懸念と92項目の勧告を発出。精神科への強制入院を可能にする法律の廃止、分離教育の中止、脱施設化の推進などを求めた。
2. **改正精神保健福祉法**（2024年4月施行）: 医療保護入院の期間上限（6ヶ月）設定、虐待通報義務の強化など。
3. **改正障害者差別解消法**（2024年4月施行）: 民間事業者にも合理的配慮の提供を義務化。
4. **発達障害者支援法成立20年**（2024年）: 2004年の議員立法成立以降、支援センターの全国整備、雇用率の拡大などが進展。

## 政党別分析

### 1. 自由民主党

**総合評価**: 部分的に一貫

**公約概要**: 障害者支援を社会保障の柱の一つと位置付け、全国どこでも安心して医療・介護・障害者福祉を受けられる体制整備を掲げる。発達障害については、スクールカウンセラーや特別支援教育支援員の配置強化を公約。情報アクセシビリティの向上も推進。

**総合分析**: 自民党は与党として障害者差別解消法（2013年成立、2021年改正）、障害者総合支援法、発達障害者支援法（2004年成立）、改正精神保健福祉法（2022年成立、2024年施行）などの立法に関与してきた実績がある。しかし、2026年衆院選の政権公約では障害者福祉に関する記述は「障害者支援」という大枠にとどまり、精神障害者や発達障害者に特化した具体的な政策は限定的である。CRPD対日審査勧告への対応についても、「現行制度の着実な実行」を基本姿勢としており、精神科強制入院の廃止など抜本的改革には慎重な立場を維持している。

#### 障害者福祉全般: 全国的な支援体制整備

**評価**: 部分的に一貫

**過去の立場**: 与党として障害者総合支援法の制定（2012年）・改正、障害者差別解消法の成立（2013年）・改正（2021年）を主導。2024年の障害福祉サービス等報酬改定では、一部サービスの基本報酬を引き下げ、障害者団体や野党から批判を受けた。

**分析**: 障害者福祉の法制度整備を推進してきた実績はあるが、2024年の報酬改定で基本報酬を引き下げたことは、「安心して福祉を受けられる体制整備」という公約との整合性に疑問が残る。J-ファイル2026（総合政策集）では情報アクセシビリティの向上など具体策も示されているが、政権公約本体での障害者福祉の記述は包括的な表現にとどまる。

#### 精神障害者政策: 現行制度の運用重視

**評価**: 部分的に一貫

**過去の立場**: 2022年の精神保健福祉法改正を主導（与党として）。医療保護入院の期間上限（6ヶ月）設定、虐待通報義務の強化を盛り込んだ。一方で、CRPD対日審査での「強制入院を可能にする法律の廃止」勧告に対しては、即座の廃止には否定的。2024年衆院選のこころのバリアフリー研究会への回答では、改正法による6ヶ月上限設定を「既に実施中」と強調し、現行制度の着実な実行に重点を置く姿勢を示した。

**分析**: 精神保健福祉法の漸進的改正は行っているが、国連勧告が求める強制入院制度の抜本的見直しには踏み込んでいない。2026年公約でも精神障害者に関する独立した政策項目はなく、現行制度の運用改善に軸足を置く姿勢は過去と一貫している。

#### 発達障害者政策: 教育支援中心のアプローチ

**評価**: 一貫

**過去の立場**: 発達障害者支援法（2004年成立）の制定に与党として関与。2016年の改正でも支援の拡充を推進。特別支援教育の充実、発達障害者支援センターの全国配置などを進めてきた。

### 2. 日本維新の会

**総合評価**: 部分的に一貫

**公約概要**: 障害者雇用の多様化（フリーランス・テレワーク推進）、介護・障害福祉従事者の処遇改善、療育発達支援施設の拡充、重度訪問介護の経済活動中利用、情報アクセス保障の推進を掲げる。

**総合分析**: 日本維新の会は社会保障制度全体の効率化・改革を基本方針としつつ、障害者雇用の多様化や福祉従事者の処遇改善など具体的な施策を提示している。2025年1月には立憲・国民と共同で「介護・障害福祉従事者の人材確保に関する特別措置法案」を国会に提出するなど、超党派での取り組みも見せている。精神障害者政策については、身体拘束を人権侵害と認めつつも全面廃止ではなく段階的削減を主張しており、現実主義的なアプローチをとっている。

#### 障害者雇用の多様化

**評価**: 一貫

**過去の立場**: 維新八策に基づく「身を切る改革」の文脈で、社会保障制度の効率化を主張してきた。障害者雇用については、法定雇用率制度に加えてフリーランスや就労継続支援事業所への発注を評価する仕組みの導入を以前から主張。

**分析**: 維新八策2026では「身体・知的・精神の障がい種別にとらわれない障がい者雇用率の向上」「分身ロボットなどの技術活用」「在宅（テレワーク）での就労系福祉サービス利用の普及」など、テクノロジー活用と多様な働き方の促進を具体的に提示しており、過去の方針と一貫している。重度訪問介護の経済活動中利用も掲げており、これは従来の維新の主張の延長線上にある。

### 3. 中道改革連合（立憲民主党＋公明党）

**総合評価**: 部分的に一貫

**総合分析**: 中道改革連合は2026年1月に立憲民主党と公明党が合流して結成された新党であり、両党の政策的伝統を引き継いでいる。立憲民主党は障害者の地域生活支援・精神科身体拘束削減・インクルーシブ教育を重視し、公明党は発達障害者支援法の成立（2004年）を主導した実績を持つ。新党の公約では障害児福祉手当の所得制限撤廃やインクルーシブ教育推進が掲げられているが、両党がそれぞれ持っていた精神障害者や発達障害者に関するより詳細な政策がどこまで引き継がれているかは、重点政策文書だけでは判断が難しい。

#### 精神障害者政策: 身体拘束削減と地域移行

**評価**: 部分的に一貫

**過去の立場（立憲民主党）**: 国連障害者権利委員会の勧告を根拠に「身体拘束の削減を進める」と表明。虐待防止法改正で第三者チェック体制構築を提案。精神科病院での身体拘束の削減、ピアサポート事業の拡充を掲げてきた。

#### 発達障害者政策: 公明党の実績を背景に

**評価**: 一貫

**過去の立場（公明党）**: 2004年、公明党の呼びかけで超党派議員連盟が発足し、発達障害者支援法を議員立法として成立させた。公明党の福島豊衆院議員（当時）が事務局長を務めた。その後も支援センター整備や通級指導の拡充を推進。2024年時点で発達障害者の雇用者数は推計9万1000人（5年間で2.3倍）。
**過去の立場（立憲民主党）**: 改正発達障害者支援法に基づく切れ目のない支援を主張。大人の発達障害への就労支援、ピアサポートの強化を掲げてきた。

**分析**: 中道改革連合の重点政策ではインクルーシブ教育の推進が明記されており、発達障害児を含む全ての子どもが共に学べる環境づくりを掲げている。これは両党の従来路線と一致している。

### 4. 国民民主党

**総合評価**: 一貫

**公約概要**: 障害者に関する公的支援全般の所得制限撤廃、発達障害の早期発見・療育体制の強化（専門職の養成・配置）、大人の発達障害への理解促進、「18歳の壁」対策法案の成立、インクルーシブ教育・雇用の推進、就労支援の充実を掲げる。

**総合分析**: 国民民主党は障害者福祉について最も具体的かつ包括的な政策を提示している政党の一つである。特に発達障害の早期発見から成人期の就労支援まで、ライフステージに応じた切れ目のない支援を体系的に示している点が特徴的である。「18歳の壁」対策法案の成立を目指すなど、具体的な立法提案も含まれている。過去の政策集と2026年公約の間には高い一貫性が見られる。

#### 精神障害者政策: 地域自立生活への移行重視

**評価**: 部分的に一貫

## 政党横断的な観察

### 1. 所得制限撤廃は超党派的合意に近づいている

障害児福祉手当や障害者に関する公的支援の所得制限撤廃は、国民民主党、中道改革連合、日本共産党、チームみらいなど複数の政党が共通して掲げている。自民党を除く多くの政党がこの方向性を支持しており、次の国会での議論が期待される。

### 2. 精神障害者政策には大きな温度差

精神障害者政策については、各政党の立場に大きな温度差がある。

| 政党 | 精神科強制入院への立場 | 身体拘束への立場 |
|------|----------------------|----------------|
| 自民党 | 現行制度の着実な運用 | 改正法に基づく対応 |
| 日本維新の会 | 廃止に同意せず | 段階的削減 |
| 中道改革連合 | 削減を推進（立憲）/現行制度運用（公明） | 削減推進 |
| 国民民主党 | 段階的改善 | 段階的改善 |
| 日本共産党 | 医療保護入院の廃止を明確に要求 | 廃止を要求 |
| れいわ新選組 | 要件厳格化、違憲・違法と主張 | 全面禁止 |
| 参政党 | 質問主意書で問題提起 | （明確な立場未確認） |

CRPD対日審査勧告が求める「強制入院を可能にする法律の廃止」に正面から応えているのは日本共産党とれいわ新選組のみである。与党（自民党・維新）は現行制度の改善を基本としており、国連勧告との間には大きな距離がある。

### 3. 発達障害者政策の充実度に差

発達障害者政策では、国民民主党が最も包括的・具体的な政策を提示している（早期発見、専門職養成、「18歳の壁」対策、成人期の就労支援まで体系的に網羅）。一方、自民党は教育分野中心、維新は療育支援施設の拡充中心と、それぞれ重点の置き方が異なる。参政党は発達障害の存在自体を否定した過去があり、当事者・家族との信頼関係に深刻な問題を抱えている。

### 4. 障害当事者の政治参加

れいわ新選組が2019年以降、重度障害者の舩後靖彦氏（ALS・2025年引退）、木村英子氏（頸椎損傷・2025年参院選で再選）、天畠大輔氏を国会に送り出したことは、日本の障害者政策の議論に大きな影響を与えた。新幹線車椅子スペースの拡大（2席から6席）や在宅避難者支援の法改正など、具体的な制度改善の実績がある。

### 5. テクノロジー活用の可能性

チームみらいがAI・デジタル技術を活用した福祉アクセス改善を提案し、日本維新の会が分身ロボットやテレワークの活用を掲げるなど、テクノロジーと障害者福祉の接点が政策議論に登場し始めている。ただし、デジタルデバイド（情報格差）への配慮が課題として残る。

### 6. 福祉従事者の処遇改善は共通課題

介護・障害福祉従事者の処遇改善は、自民党、維新、中道改革連合、共産党、チームみらいなど多くの政党が共通して掲げている。2025年1月には維新・立憲・国民が共同で「介護・障害福祉従事者の人材確保に関する特別措置法案」を国会に提出しており、超党派での取り組みも見られる。
