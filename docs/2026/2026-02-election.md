# 2026年2月 第51回衆議院議員総選挙 総合レポート

**選挙日**: 2026年2月8日投開票
**生成日時**: 2026-02-08

## 凡例

- 🟢 一貫: 過去の立場と一貫している
- 🟡 部分的に一貫: 部分的に一貫しているが、変更や修正がある
- 🔴 矛盾: 過去の立場と矛盾している
- 🔵 新規: 過去に明確な立場がなく、新しい主張である

---

## 1. 選挙概要

2026年2月8日投開票の第51回衆議院議員総選挙。衆議院の全465議席が争われる。高市早苗首相（日本初の女性首相、2025年10月就任）が通常国会で衆議院を解散し、短期決戦となった。高市首相は「与党で過半数を取れなければ即座に辞任する」と表明している。

---

## 2. 政局の大きな変化

### 2.1 自公連立の解消と新たな枠組み

- **2025年10月**: 公明党が裏金問題や高市氏の路線をめぐる対立から、26年間続いた自公連立を解消
  - 連立解消の理由: 企業・団体献金の規制強化、裏金事件の全容解明、靖国神社参拝問題、外国人との共生問題への対応をめぐる対立
- **2025年10月20日**: 自民党は日本維新の会と新たに連立を組む（「日本再起への12本の矢」を合意）
- **2026年1月16日**: 立憲民主党と公明党が衆議院限定で合流し、新党「**中道改革連合**」（略称：中道、約172人）を結成

### 2.2 連立再編の影響

自公連立から自維連立への転換は、日本の政治に大きな影響を与える可能性がある。公明党は長年、自民党の右寄りの政策に対する「歯止め役」を果たしてきたが、維新はより自民党に近い立場を取ることが多い。特に憲法改正、靖国神社参拝、外国人政策、教育政策などの分野で政策の右傾化が進む可能性がある。

---

## 3. 主な争点

1. **物価高対策・消費税減税** — ほぼ全政党が何らかの消費税減税を掲げる（チームみらいを除く）
2. **外国人政策** — 不動産取得規制など
3. **社会保険料の負担軽減**
4. **政治改革** — 議員定数削減、企業・団体献金規制
5. **安全保障・防衛政策**

## 4. 各政党の公約と一貫性分析

### 4.1 自由民主党（連立：維新と）

**総合評価**: 🔴 矛盾

**主な公約**:
- 食料品の消費税を**2年間ゼロ**に
- 危機管理や成長投資のための「新たな予算枠」
- 衆議院議員定数の**1割削減**を次期国会で法案成立目標
- 連立合意に基づく「12本の矢」

#### 🔴 食料品の消費税を2年間ゼロに

自民党は長年にわたり「社会保障の安定財源」として消費税の重要性を主張し、増税を推進してきた。2012年の三党合意、2014年の8%引き上げ、2019年の10%引き上げはいずれも自民党政権下で実施された。2022年から2024年まで、党幹部は一貫して消費税減税に反対してきた。

高市首相自身も、2025年9月の総裁選では消費税減税の主張を事実上封印し、首相就任後も「即効性がない」として慎重姿勢を示していたが、2026年1月の衆院解散表明で突如として方針転換した。過去30年以上にわたって消費税増税を推進してきた党の立場とは根本的に矛盾している。

#### 🔴 衆議院議員定数の1割削減

自民党は2010年代から繰り返し議員定数削減を公約に掲げながら、実行してこなかった。2012年の三党合意では安倍総裁が明確に約束したにもかかわらず、政権復帰後に実行しなかった。2025年1月には結論を「2026年以降に」先送りすることで合意していたにもかかわらず、数か月後の公約では「次期国会で法案成立を目指す」と前倒しして掲げている。過去14年間にわたって約束を反故にしてきた実績を考えると、一貫性は皆無である。

#### 🔵 「新たな予算枠」

「責任ある積極財政」のもと、経済安全保障、食料、エネルギーなどに戦略的に投資する「新たな予算枠」は新規の政策だが、長年掲げてきた「PB黒字化」目標との整合性に疑問がある。食料品消費税ゼロ化で年間約5兆円の税収減が見込まれる中、さらなる大規模支出の財源は不明確。

#### 🔴 維新との連立（自公連立の解消）

26年間続いた自公連立を解消し、裏金問題への対応を回避する形で公明党を切り、維新と組んだことは、党の立場の大きな転換である。公明党は企業・団体献金の規制強化と事件の全容解明を求めたが、自民党はこれに応じなかった。

### 4.2 日本維新の会（連立：自民と）

**総合評価**: 🟡 部分的に一貫

**主な公約**:
- 食料品の消費税2年間ゼロ（自民と共通）
- 国民医療費を年間4兆円以上削減、現役世代の社会保険料を年間6万円引き下げ
- 「**副首都構想**」— 大阪・福岡・札幌などを候補に副首都法を制定
- 衆院議員定数1割削減、企業・団体献金の見直し
- 反撃能力として長射程ミサイル配備、「専守防衛」の解釈見直し

#### 🟡 食料品の消費税2年間ゼロ

消費税減税という方向性は一貫しているが、2017年の「消費税の地方税化」、2024年の「段階的引き下げ後に5%」から、連立合意に基づく「食料品2年間ゼロ」へと具体的手法は変遷している。

#### 🔴 企業・団体献金の「見直し」

2024年衆院選では「企業・団体献金の全面禁止」を明確に掲げていたが、2026年では「見直し」と曖昧な表現にトーンダウン。連立協議で議員定数削減を前面に押し出し、企業献金問題を棚上げした。

#### 🟢 衆院議員定数1割削減 / 専守防衛の解釈見直し

議員定数削減は創設以来の核心的政策であり一貫。安全保障政策の強化も従来からの主張の延長線上にある。

#### 🟡 副首都構想

2度の住民投票で否決された大阪都構想を「副首都構想」という名称で実質的に復活。対象を全国に広げた点は新しいが、住民投票の結果との整合性は評価が分かれる。

#### 🔴 自民党との連立

2024年3月には「与党を過半数割れに追い込む」ことを目標に掲げ、自民党の裏金問題を批判していたが、わずか1年後に連立に踏み切った。維新の存在意義そのものが問われている。

### 4.3 中道改革連合（立憲民主党＋公明党、約172人）

**総合評価**: 🔴 矛盾

**主な公約**:
- 食料品消費税の**恒久的ゼロ**（今秋から実施）
- 若者・学生向け賃貸住宅の家賃補助・住宅提供
- 中道路線を掲げた政権交代を目指す

#### 🔴 食料品消費税の恒久的ゼロ（立憲民主党側）

野田佳彦代表は2012年に消費税10%への引き上げを「政治生命を賭けて」実現し、2024年の代表選でも「安易な消費税減税はしない」と明言していた。しかし党内の減税派の圧力と選挙対策により、2025年4月に「時限的ゼロ」、2026年1月には「恒久的ゼロ」へと拡大。明確な矛盾が認められる。

財源として「ジャパンファンド」構想（政府保有資産500兆円の運用利回り1%改善で5兆円捻出）を提示しているが、実現可能性に疑問が残る。

#### 🟡 食料品消費税の恒久的ゼロ（公明党側）

軽減税率の「恒久的措置」という方針は一貫しているが、水準が「8%→5%→0%」へと急速に拡大。2025年5月時点では「5%程度」を想定していた。

#### 🔴 自公連立の解消と立憲民主党との合流

公明党が26年間の自公連立を解消し、わずか3か月後に政敵であった立憲民主党と合流したことは、支持者にとって理解困難な転換。両党の「中道」の定義も異なり（立憲: 政治スペクトラムの中間、公明: 人間中心主義）、「生活者ファースト」という曖昧な概念で覆い隠していると指摘される。

#### 🟡 中道路線への転換

野田代表個人の立場とは整合的だが、立憲民主党全体としては明確な路線転換。党内の2割程度を占める「リベラル派」からの抵抗が残る。

### 4.4 国民民主党

**総合評価**: 🟢 一貫

**主な公約**:
- 消費税を一律**5%**に引き下げ（賃金上昇率が物価+2%に安定するまで）
- 基礎控除を**178万円**に引き上げ（年収の壁問題の解消）
- 社会保険料の還付制度創設
- 年間5兆円の「教育国債」発行、高校までの教育費完全無償化
- 家賃控除制度の新設、「空室税」導入

#### 🟢 消費税を一律5%に引き下げ

2019年頃から一貫して消費税減税を主張。2021年衆院選以降は「賃金上昇率が物価+2%に達するまで消費税5%」を継続的に掲げている。2025年7月に「今の賃上げ水準なら不要」→翌日「やるべき」と発言が揺れたが、条件重視の姿勢の表れと解釈できる。

#### 🟢 基礎控除を178万円に引き上げ

2024年衆院選で初めて「178万円」を明確に掲げ、2025年末に自民党との政策協議で実現。政策の継続性と実現力を示した。

#### 🔵 社会保険料の還付制度 / 家賃控除・空室税

2024年以降に新たに追加された政策だが、「手取りを増やす」という一貫した理念に基づいており、政策体系の拡充と見なせる。空室税は2025年12月に議員立法として法案提出済み。

#### 🟢 教育国債

2021年衆院選以降一貫して公約に掲げ、2022年には議員立法で法案を提出。「毎年5兆円」の数値目標も一貫。

### 4.5 日本共産党

**総合評価**: 🟢 一貫

**主な公約**:
- 消費税廃止を目指し、緊急に**5%**に引き下げ
- 最低賃金1,700円に引き上げ
- 大企業の内部留保課税で5年間10兆円の財源創出
- 防衛費増額反対、安保関連法廃止、核兵器禁止条約参加
- 2040年度までに再エネ100%、原発再稼働反対
- 選択的夫婦別姓・同性婚の早期実現

#### 🟢 消費税廃止

1989年の消費税創設以来37年間、一貫して反対。2021年以降は「緊急に5%」という段階的アプローチを継続。

#### 🟡 最低賃金1,700円

目標額が1,500円→1,700円に引き上げられたが、物価高騰に対応した調整であり基本方針の変更ではない。

#### 🟢 内部留保課税

2022年以降、課税率（年2%、5年間で計10%）、対象、財源規模すべて同一。「二重課税」批判はあるが主張は一貫。

#### 🟡 安保・防衛政策

安保条約廃棄、安保関連法廃止は一貫。ただし自衛隊について、理論的には「違憲」としつつ政権参加時には「活用する」という二重の立場がある。

#### 🟢 原発反対 / 選択的夫婦別姓・同性婚

原発反対は1961年以来65年間完全に一貫。選択的夫婦別姓は1996年以降30年近く、同性婚は2019年以降継続的に主張。

### 4.6 れいわ新選組

**総合評価**: 🟢 一貫

**主な公約**:
- 消費税の**完全廃止**
- 物価高対策として現金**10万円**の一律給付

#### 🟢 消費税の完全廃止

2019年の結党時から7年間、一度も揺らぐことなく主張。「5%に減税」ではなく「完全廃止」とする点も終始一貫。

#### 🟢 現金10万円の一律給付

2020年のコロナ禍以降、継続的に主張。所得制限を設けない普遍的給付という方針も結党時からの理念と一致。MMT（現代貨幣理論）に基づく積極財政論が理論的基盤。

2026年の公約は「消費税廃止」と「10万円給付」の2点に絞り込まれているが、最低賃金1500円、奨学金徳政令、住宅政策などは党の基本政策として維持されており、選挙戦略としてのメッセージング絞り込みと考えられる。

### 4.7 参政党

**総合評価**: 🟢 一貫

**主な公約**:
- 0〜15歳の子ども1人あたり月**10万円**の「教育給付金」
- 外国人による不動産取得の規制

#### 🟢 月10万円の教育給付金

2022年参院選から一貫して主張し、4年間変更なし。2025年参院選では「日本国籍を有する家庭を優先対象」という条件が明示された。ただし必要予算は年間約17.9兆円と試算され、「教育国債」という財源説明にとどまり、具体的な返済計画は不透明。

#### 🟢 外国人による不動産取得の規制

結党当初から一貫。WTO/GATS協定での日本の不平等な署名状況を問題視し、外国人総合政策庁の設置、受け入れ総量の厳格化を提起。産経調査では国民の77.2%が規制に賛成。

### 4.8 チームみらい

**総合評価**: 🟢 一貫

**主な公約**:
- 社会保険料の引き下げ（最重要政策）
- 子育て減税
- AI・ロボット・自動運転などの新産業への投資
- 政治資金の透明化（「みらいまる見え政治資金」ツール）
- 消費税率は維持（全政党中ほぼ唯一）

2025年5月設立、2025年7月の参院選で73日間での国政政党化（日本史上最速）。AIエンジニアの安野貴博党首（35歳）が率いる。

#### 🟢 テクノロジーによる政治改革 / 現役世代への焦点

結党時から「テクノロジーで、誰も取り残さない日本をつくること」を掲げ、一貫して社会保険料削減を優先。消費税減税ではなく社会保険料削減を重視する独自路線は結党初期から変わっていない。

都知事選（コンセプト実証）→ 参院選（政策の体系化）→ 衆院選（具体化と優先順位の明確化）という「アジャイル的」な政策進化が特徴。

## 5. 政党横断テーマ分析

### 5.1 消費税政策の比較

2026年衆院選の特徴は、ほぼ全政党が何らかの消費税減税を公約に掲げている点である。これは過去に例のない現象。

| 政党 | 消費税政策 | 特徴 | 過去との一貫性 |
|------|-----------|------|--------------|
| 自民党・維新 | 食料品2年間ゼロ | 期間限定・品目限定 | 🔴 矛盾（自民）/ 🟡 部分的一貫（維新） |
| 中道改革連合 | 食料品恒久的ゼロ | 恒久的・品目限定 | 🔴 矛盾（立憲）/ 🟡 部分的一貫（公明） |
| 国民民主党 | 一律5% | 条件付き・全品目 | 🟢 一貫 |
| 共産党 | 緊急5%、将来廃止 | 段階的・全品目 | 🟢 一貫 |
| れいわ新選組 | 完全廃止 | 即時・全品目・恒久的 | 🟢 一貫 |
| チームみらい | **維持** | 社保料削減を優先 | 🟢 一貫 |

自民党が消費税増税路線を放棄し減税に転じたことは、日本の財政政策における転換点となる可能性がある。

### 5.2 財源問題の不透明性

各党とも大規模な支出・減税を掲げているが、財源の具体性には差がある。

| 政党 | 主な財政負担 | 財源説明 |
|------|------------|---------|
| 自民党 | 消費税ゼロ（約5兆円/年）＋新予算枠＋社保料引き下げ | 「税外収入も活用」（曖昧） |
| 中道改革連合 | 食料品消費税ゼロ（約5兆円/年） | 「ジャパンファンド」（実現性に疑問） |
| 国民民主党 | 消費税5%＋教育国債5兆円/年 | 教育国債、日銀保有国債の永久国債化、特会活用 |
| 共産党 | 消費税5%＋各種無料化 | 大企業・富裕層への課税強化で30兆円 |
| れいわ新選組 | 消費税廃止＋10万円給付 | 国債発行（MMT理論） |
| 参政党 | 教育給付金（約17.9兆円/年） | 教育国債 |

### 5.3 政治改革 — 企業献金と議員定数

議員定数削減は2010年代から繰り返し公約に掲げられながら、ほとんど実現してこなかった「空約束」の典型例である。

自公連立解消の最大の要因は「政治とカネ」の問題だったが、自民党は企業献金規制に応じず、維新も「全面禁止」から「見直し」に後退した。政治改革を求める声と実際の対応の乖離が際立つ。

### 5.4 障害者福祉・障害者の権利

#### 精神障害者政策の温度差

| 政党 | 精神科強制入院への立場 | 身体拘束への立場 |
|------|----------------------|----------------|
| 自民党 | 現行制度の着実な運用 | 改正法に基づく対応 |
| 日本維新の会 | 廃止に同意せず | 段階的削減 |
| 中道改革連合 | 削減を推進（立憲）/現行制度運用（公明） | 削減推進 |
| 国民民主党 | 段階的改善 | 段階的改善 |
| 日本共産党 | 医療保護入院の廃止を明確に要求 | 廃止を要求 |
| れいわ新選組 | 要件厳格化、違憲・違法と主張 | 全面禁止 |

CRPD対日審査勧告が求める「強制入院を可能にする法律の廃止」に正面から応えているのは共産党とれいわ新選組のみ。

#### 発達障害者政策

国民民主党が最も包括的・具体的（早期発見、専門職養成、「18歳の壁」対策、成人期の就労支援まで体系的に網羅）。参政党は党の公式出版物で「発達障害など存在しない」と記載した過去があり、日本自閉症協会等から厳しい批判を受けた。

#### 所得制限の撤廃

障害児福祉手当や公的支援の所得制限撤廃は、国民民主党、中道改革連合、共産党、チームみらいなど複数政党が共通して掲げており、超党派的合意に近づいている。

#### 障害当事者の政治参加

れいわ新選組が2019年以降、重度障害者の舩後靖彦氏、木村英子氏、天畠大輔氏を国会に送り出し、新幹線車椅子スペースの拡大（2席→6席）や重度訪問介護の利用制限問題の可視化など、具体的な制度改善を実現。
