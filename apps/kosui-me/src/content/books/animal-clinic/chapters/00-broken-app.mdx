```bash
git checkout step-00-end
pnpm install
pnpm test
```

テストはすべて通ります。要件は最低限満たされています。

## 素朴にエージェントに依頼してみる

本章は執筆中です。AIエージェントへの依頼プロンプトと、生成された素朴な実装の要点を掲載予定です。

## 何が痛いのか

- 単一の `JobApplication` 型に `status: string` と全フィールドを詰め込んでいる
- `throw` だらけ、`as JobApplication` だらけ、巨大関数
- ロガーに応募者をまるごと渡している

これらは「動いてしまう」ため一見問題なく見えますが、次の要件が降ってきた瞬間に破綻が広がります。
