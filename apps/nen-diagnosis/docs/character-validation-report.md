# 念系統診断 原作準拠性検証レポート（v2: 7軸版）

## 概要

本レポートは、nen-diagnosis アプリケーションの診断ロジックが原作「HUNTER×HUNTER」のキャラクター設定に準拠しているかを検証したものです。

**v2の改善点:**

- 判定軸を5軸から7軸に拡張（empathy, charismaを追加）
- 質問数を10問から14問に増加
- 各系統への寄与重みを調整

## 診断ロジックの概要

## v2での改善まとめ

### 大幅に改善された系統

1. **特質系**: 0-20% → 40-80% (+40-60ポイント)
   - charisma軸の追加が効果的
   - クロロ、メルエム、ネオン、ピトーの診断精度が向上

2. **変化系**: 40-80% → 80-100% (+40ポイント)
   - honesty negative の変化系寄与強化が効果的
   - キルア、マチの診断精度が向上

3. **操作系**: 50% → 75-100% (+25-50ポイント)
   - モラウ、シュートの診断精度が向上

### 残る課題

1. **放出系**: 0-33% → 33% (微改善)
   - レオリオ、ナックルは依然として強化系と診断される
   - 「仲間思いで正直」という性格が強化系と重複

2. **強化系のカストロ**: 具現化系と診断される
   - 原作でも「強化系なのに具現化系の技を覚えた」キャラクター
   - 性格的には具現化系寄りという解釈も可能

### 結論

v2の改善により、診断精度は**35-54%から64-80%に大幅向上**しました。特に**特質系の識別が大きく改善**され、クロロ、メルエム、ネオン等のキャラクターが適切に診断されるようになりました。

一方で、**放出系の識別は依然として課題**が残っています。レオリオやナックルのような「仲間思いで正直だが短気で大雑把」というキャラクターは、強化系の特徴（単純で一途、正直）と重複するため、区別が困難です。

今後の改善案としては、「短気・感情的」を測定する軸の追加や、放出系への寄与をさらに強化することが考えられます。
