# 念系統診断 原作準拠性検証レポート

## 概要

本レポートは、nen-diagnosis アプリケーションの診断ロジックが原作「HUNTER×HUNTER」のキャラクター設定に準拠しているかを検証したものです。

## 診断ロジックの概要

### 判定軸から念系統への寄与マッピング

| 軸           | positive側     | → 念系統             | negative側       | → 念系統             |
| ------------ | -------------- | -------------------- | ---------------- | -------------------- |
| commitment   | 一途・こだわり | 強化(1.0), 具現(0.5) | 柔軟・適応       | 変化(1.0), 放出(0.5) |
| logic        | 論理的・計画的 | 操作(1.0), 具現(0.5) | 直感的・即興     | 放出(1.0), 変化(0.5) |
| independence | 個人主義・独自 | 特質(1.0), 変化(0.5) | 協調的・順応     | 強化(1.0), 放出(0.5) |
| caution      | 慎重・几帳面   | 具現(1.0), 操作(0.5) | 大胆・大雑把     | 放出(1.0), 強化(0.5) |
| honesty      | 率直・正直     | 強化(1.0), 放出(0.5) | 策略的・計算高い | 操作(1.0), 変化(0.5) |

## 問題点の詳細分析

### 1. 放出系の識別困難

**現状の問題:**

- 放出系キャラクター（レオリオ、ナックル）が全員「強化系」と診断される
- 原作設定「短気で大雑把、仲間思い」のうち、「仲間思い」が強化系に寄与してしまう

**原因:**

```
independence(negative/協調的) → 強化系(1.0), 放出系(0.5)
honesty(positive/正直) → 強化系(1.0), 放出系(0.5)
```

放出系への寄与が常に強化系の半分しかない。

### 2. 特質系の識別困難

**現状の問題:**

- 特質系キャラクター5名中、4名が完全に不一致
- 特質系への寄与経路が`independence`軸のみ

**原因:**

```
independence(positive/個人主義) → 特質系(1.0), 変化系(0.5)
```

特質系の「カリスマ性」「普通とは違う視点」を測定できていない。

### 3. 変化系と放出系の混同

**現状の問題:**

- ヒソカ、キルア、フェイタンが変化系と放出系の同率になる

**重複する寄与元:**

- `commitment(negative)`: 変化系(1.0), 放出系(0.5)
- `logic(negative)`: 放出系(1.0), 変化系(0.5)
- `caution(negative)`: 放出系(1.0), 強化系(0.5)

---

## 改善提案

### 提案1: 新しい判定軸の追加

**「情の深さ」軸（empathy）**

- positive(情に厚い) → 放出系(1.0), 強化系(0.5)
- negative(冷淡) → 操作系(1.0), 特質系(0.5)

**「カリスマ性」軸（charisma）**

- positive(人を惹きつける) → 特質系(1.0), 強化系(0.5)
- negative(目立たない) → 具現化系(1.0), 操作系(0.5)

### 提案2: 既存軸の重み調整

**honesty軸の調整:**

```
現在: negative → 操作系(1.0), 変化系(0.5)
提案: negative → 操作系(0.8), 変化系(1.0)
```

**independence軸の調整:**

```
現在: positive → 特質系(1.0), 変化系(0.5)
提案: positive → 特質系(1.5), 変化系(0.5)
```
