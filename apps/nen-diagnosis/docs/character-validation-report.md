# 念系統診断 原作準拠性検証レポート（v3: 7軸強化版）

## 概要

本レポートは、nen-diagnosis アプリケーションの診断ロジックが原作「HUNTER×HUNTER」のキャラクター設定に準拠しているかを検証したものです。

**v3の改善点:**

- caution軸を「慎重さ」から「几帳面さ」に名称変更
- Q7・Q8の質問を「几帳面/大雑把」をより明確に測定する内容に改善
- caution軸negative側の放出系への寄与を1.0から1.5に強化
- caution軸negative側から強化系への寄与(0.3)を削除

**v2の改善点:**

- 判定軸を5軸から7軸に拡張（empathy, charismaを追加）
- 質問数を10問から14問に増加
- 各系統への寄与重みを調整

## 診断ロジックの概要

## v3での改善まとめ

### v3で行った変更

1. **caution軸の強化**
   - 「慎重さ」→「几帳面さ」に名称変更
   - Q7: 「書類やメールの誤字脱字確認」という具体的な質問に改善
   - Q8: 「計画を立てるときの細かさ」という具体的な質問に改善
   - negative側の放出系寄与: 1.0 → 1.5 に増加
   - negative側の強化系寄与: 0.3 → 0 に削除

### v3での効果

1. **放出系キャラクターのスコア改善**
   - フランクリン: 放出系優位がさらに強化（差+0.3→+1.9）
   - ナックル: 強化系との差が縮小（2.6→1.0）
   - レオリオ: 放出系スコア増加（4.4→5.4）

2. **副次的な影響**
   - モラウ: 操作系→放出系に変化（caution negativeの影響）
   - ピトー: 2位が強化系→放出系に変化

### 残る課題

1. **放出系のレオリオ・ナックル**: 依然として強化系と診断
   - 「仲間思いで正直」という性格が強化系と重複
   - 「短気・感情的」を測定する軸の追加が有効

2. **強化系のカストロ**: 具現化系と診断
   - 性格的には具現化系寄り（原作でも念の系統選択ミスのキャラ）

### 結論

v3の「几帳面さ」軸強化により、**放出系キャラクターの識別精度が向上**しました。特に**フランクリンの診断信頼性が大幅に向上**し、ナックルやレオリオも放出系スコアが上昇しています。

ただし、**レオリオとナックルは依然として強化系と診断**されます。これは「仲間思いで正直」という性格特性が強化系に強く寄与するためです。原作の放出系設定を正確に再現するには、**「短気・感情的」「衝動的」を測定する軸の追加**が最も有効と考えられます。

### 今後の改善案

1. **temperament（気質）軸の追加**
   - positive: 「短気・衝動的」→ 放出系
   - negative: 「冷静・忍耐強い」→ 操作系、具現化系

2. **放出系への寄与をさらに特化**
   - empathy軸の放出系寄与を強化
   - independence軸negative（協調的）の放出系寄与を追加
